String Evolution! 
using Genetic Algorithms!

The program takes a string from user and evolves random gibberish into the string using genetic selection try it out yourself ! download the zip and run the exe directly nothing else!

okay so lets break it down : 
we have a few major checkpoints to build for our genetic algorithm

1. **Genetic Representation of Solution**: We have to make whatever problem we have into a genome-population type genetic format (i.e representing in that way)
2. **Generate New Functions**:  Now we need a function to generate many solutions which may be wrong or correct
3. **Fitness Function** : This function is like the corrector it tells the algorithm what is right and whats not working
4. **CrossOver Function** : This is mixing and matching to create new stuff which may or may not be better (it mostly is)
5. **Mutation Function** : This is the real deal its what makes shit happen, new patterns! which wouldnt have come otherwise from crossovers although its very unstable you may lose results with bad values.

The whole thing mimics biology and how many things have come to be by evolving and mutating to the surroundings. This whole things recursives itself into solving biology problems themselves!
